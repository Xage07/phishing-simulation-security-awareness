# phishing-simulation-security-awareness

## Project Overview
This project documents a controlled phishing simulation carried out to assess user susceptibility to phishing attacks in a simulated organizational environment (TechCrush).

## Objective
To evaluate phishing awareness, measure click-through rates, and demonstrate ethical offensive security testing.

## Tools & Technologies
- Kali Linux
- Gophish
- MailHog
- VirtualBox

## Methodology
1. Environment setup
2. Phishing campaign creation
3. Email template and landing page design
4. Campaign execution
5. Result analysis

## Results
The campaign produced measurable interaction metrics which were analyzed and documented in the technical report.

## Ethical Considerations
- No real users were harmed
- All data anonymized
- Project conducted for educational purposes only
